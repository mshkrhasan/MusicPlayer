<h1>🎵 Music Player</h1>

<p>
A premium Android Music Player built with <strong>Media3 (ExoPlayer)</strong>,
featuring high-fidelity audio playback, video-to-audio support, dynamic Material 3 UI,
playlists, folder browsing, and seamless background playback.
</p>

<hr>

<h2>✨ Features</h2>

<h3>🎵 Core Playback Features</h3>

<ul>
<li>
<strong>High-Fidelity Audio Engine</strong>
<ul>
<li>Powered by Google Media3 (ExoPlayer)</li>
<li>Smooth, reliable, and low-latency playback</li>
</ul>
</li>

<li>
<strong>Video-to-Audio Playback</strong>
<ul>
<li>Play audio from video files in the background</li>
<li>Perfect for music videos, lectures, podcasts, and recordings</li>
</ul>
</li>

<li>
<strong>Intelligent Media Resumption</strong>
<ul>
<li>Automatically remembers the last played song</li>
<li>Saves playback queue and position</li>
<li>Resume listening exactly where you left off</li>
</ul>
</li>

<li>
<strong>System-Wide File Support</strong>
<ul>
<li>Open audio and video files directly from File Manager</li>
<li>Supports Android file sharing and media intents</li>
</ul>
</li>
</ul>
